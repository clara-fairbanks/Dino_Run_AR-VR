# Dino_Run_AR-VR
Game developed in Unity3d using Vuforia for Augmented Reality 
Jungle is adaptable for Google Cardboard Virtual Reality
To run, clone repository and use attached instructions to build from unity.
To contribute, add a scene or fix a bug, clone the repo, make your awesome changes, and submit a pull request.
To contribute by suggestion submit an issue or make a comment!
